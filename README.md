# Sistem Stok Barang Koperasi

Sistem manajemen stok barang untuk koperasi berbasis PHP.

## Fitur
- Manajemen Barang (Stok)
- Barang Masuk
- Barang Keluar
- Laporan Stok
- Log Aktivitas
- Pengaturan Sistem

## Instalasi
1. Clone repositori ini.
2. Impor database dari folder `database/sistem_koperasi_stok.sql`.
3. Sesuaikan konfigurasi database di `config/koneksi.php`.
4. Jalankan di server lokal (seperti Laragon atau XAMPP).
